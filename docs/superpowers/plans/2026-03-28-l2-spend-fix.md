# L2 LLM Spend Fix — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stop the `/api/agent/l2/plan` endpoint from burning LLM credits on duplicate and non-remediable drift findings. Reduce 1,251 LLM calls/week to ~20.

**Architecture:** Three server-side defenses — (1) reject monitoring-only checks before they reach the LLM, (2) cache L2 decisions by pattern_signature so identical drift findings reuse prior answers, (3) fix budget tracker pricing from Haiku to Sonnet 4. All changes are backend-only; no daemon deploy required.

**Tech Stack:** Python/FastAPI backend, asyncpg via SQLAlchemy, Go daemon budget tracker (pricing fix only)

When the daemon sends the same drift finding (e.g., `backup_verification` on host .250) every 15 minutes, the LLM answer is identical every time. Cache L2 decisions by `pattern_signature` with a 24h TTL. If a cached decision exists, return it without calling the LLM.

The `pattern_signature` is a hash of `(incident_type + check_type + key details)` — already computed by the daemon and sent in the request. We query `l2_decisions` for a recent match.

```python
async def lookup_cached_l2_decision(
    db,
    pattern_signature: str,
    incident_type: str,
    ttl_hours: int = 24,
) -> Optional[Dict[str, Any]]:
    """Look up a recent L2 decision for the same pattern_signature.

    Returns the cached decision as a dict if found within ttl_hours,
    or None if no cache hit. This avoids calling the LLM for identical
    drift findings that recur every scan cycle.
    """
    if not pattern_signature:
        return None

    from sqlalchemy import text

    result = await db.execute(text("""
        SELECT runbook_id, reasoning, confidence, pattern_signature,
               llm_model, llm_latency_ms, requires_human_review
        FROM l2_decisions
        WHERE pattern_signature = :sig
        AND created_at > NOW() - INTERVAL :ttl_hours
        ORDER BY created_at DESC
        LIMIT 1
    """), {
        "sig": pattern_signature,
        "ttl_hours": f"{ttl_hours} hours",
    })

    row = result.fetchone()
    if row is None:
        return None

    return {
        "runbook_id": row[0],
        "reasoning": row[1],
        "confidence": row[2],
        "pattern_signature": row[3],
        "llm_model": row[4],
        "llm_latency_ms": row[5],
        "requires_human_review": row[6],
    }
```

    # L2 decision cache: if we already analyzed this exact pattern recently, reuse it
    from dashboard_api.l2_planner import lookup_cached_l2_decision
    cached = None
    if request.pattern_signature:
        try:
            cached = await lookup_cached_l2_decision(db, request.pattern_signature, request.incident_type)
        except Exception as e:
            logger.warning(f"L2 cache lookup failed (proceeding to LLM): {e}")

    if cached:
        logger.info("L2 cache hit — reusing prior decision",
                     site_id=request.site_id,
                     incident_type=request.incident_type,
                     pattern_signature=request.pattern_signature,
                     cached_runbook=cached["runbook_id"])

        action = "escalate"
        action_params = {}
        escalate = True

        if cached["runbook_id"] and cached["confidence"] >= 0.6:
            escalation_only_runbooks = {"RB-CERT-001", "RB-DISK-001"}
            is_escalation = (
                cached["runbook_id"] in escalation_only_runbooks
                or cached["runbook_id"].startswith("ESC-")
            )
            if is_escalation:
                action = "escalate"
                escalate = True
            else:
                action = "execute_runbook"
                escalate = False
            action_params = {"runbook_id": cached["runbook_id"]}

        return {
            "incident_id": request.incident_id,
            "recommended_action": action,
            "action_params": action_params,
            "confidence": cached["confidence"],
            "reasoning": cached["reasoning"],
            "runbook_id": cached["runbook_id"] or "",
            "requires_approval": cached["requires_human_review"],
            "escalate_to_l3": escalate,
            "context_used": {
                "llm_model": cached["llm_model"],
                "llm_latency_ms": 0,
                "pattern_signature": cached["pattern_signature"],
                "alternative_runbooks": [],
                "cache_status": "cached_24h",
