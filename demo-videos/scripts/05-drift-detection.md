## Scene 1: Opening

[SCREEN: Navigate to site detail showing drift categories]

Compliance isn't a one-time checkbox. Configurations drift. Services stop. Patches get missed. OsirisCare scans for forty-seven different drift types across every device, every ten minutes.

## Scene 2: Scan Categories

[SCREEN: Mouse over the different compliance categories — patching, firewall, backup, etc.]

We check everything that matters for HIPAA. Firewall rules, encryption status, backup health, logging configuration, antivirus definitions, patch levels, SSH hardening, screen lock policies — across Windows, Linux, macOS, and network devices. Each check maps directly to a HIPAA control.

## Scene 3: Live Detection

[SCREEN: Show a drift event appearing in real-time or recently detected]

Watch this — a firewall rule was modified on one of the workstations. The appliance detected it within ten minutes, flagged it as a compliance drift, and it's already been routed through the healing pipeline. By the time you're reading this alert, the fix is already in progress.

## Scene 4: Per-Site Configuration

[SCREEN: Navigate to drift configuration settings]

And every site is different. A dental practice has different needs than a hospital network. You can toggle individual checks per site — disable what doesn't apply, tighten what matters most. The compliance score automatically adjusts to reflect only the checks that are active.

## Scene 5: Close

[SCREEN: Return to compliance score showing improvement]

Continuous detection. Automatic remediation. Evidence captured at every step. Your compliance posture doesn't degrade between audits — it improves continuously.
